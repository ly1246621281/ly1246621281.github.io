链表整理

单向链表节点

0.建链表（尾插、头插）

1.反转链表

快慢指针，快指针比慢指针快1，局部反转。注意别循环溢出。

2.使用快慢指针来找到链表的中点

3.单链表-快慢指针（是否有环，环入口）

4.删除链表的倒数第n个节点

快慢指针，快指针指向null时，慢指针指向倒数第n+1个节点，即可卸载第n个节点。

快指针比慢指针快n+1个。

1 2 3 4 5 6 null

倒数第二个 则找到4（快3个，快指针指向null，慢指针为4）

5.判断是否为回文链表

获取中值节点，头半段反转，与后续low指针指向链表比对。
